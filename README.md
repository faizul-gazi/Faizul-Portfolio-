# Gazi Faizul Islam | Front-End React.js Developer Portfolio 🚀

Welcome to the official source code of my personal developer portfolio website. This is a highly polished, responsive, and performance-optimized single-page web application designed using modern web guidelines and premium design aesthetics.

Live Demo: [faizul-portfolio-fftz.vercel.app](https://faizul-portfolio-fftz.vercel.app/)

---

## 🎨 Design System & Visuals

This portfolio is built with premium aesthetics in mind, featuring:
* **Vibrant Accent Colors:** Tech-inspired HSL palettes using Fuchsia, Indigo, and Cyan.
* **Dual Theme Toggle:** Instant switching between a clean, off-white Slate-50 Light Theme and a deep obsidian Dark Theme, with local storage persistence and automated cache overriding.
* **Mock Browser Showcases:** Projects are housed inside realistic mock browser frames (complete with URL address bars and window action buttons) sitting on top of custom, alternating SVG vector curve backgrounds.
* **Interactive Timelines & Tabs:** Smooth interactive vertical tab lists for work experience and fully responsive skill category columns.
* **Fluid Micro-Animations:** Dynamic cursor hover effects, sliding indicators, smooth section scroll spy navigation, and fade-in transitions.

---

## 🛠️ Technology Stack

* **Core Framework:** [React.js](https://react.dev/) (v19)
* **Build Tool:** [Vite](https://vite.dev/) (v6)
* **Styling Engine:** [Tailwind CSS v4](https://tailwindcss.com/) (using native CSS variables theme configuration)
* **Icons:** [FontAwesome](https://fontawesome.com/) (loaded via parallel preconnected CDN links)
* **Fonts:** [Sora](https://fonts.google.com/specimen/Sora) (Headings) & [Inter](https://fonts.google.com/specimen/Inter) (Body text) from Google Fonts

---

## ✨ Features

* **Home (Hero):** Sleek, animated typewriter effect showcasing key specializations with a high-contrast liquid profile photo container.
* **Projects:** Alternating grid mockups with zoomable lightbox detail galleries and circled direction links (`↗`).
* **Experience & Skills:** Interactive job role tabs with smooth `fadeInUp` slide transitions and skill categories loaded with custom brand logo vector SVGs.
* **Services:** Clean presentation of technical skills and value offerings.
* **About:** Minimalist, clear professional bio statement.
* **Testimonials:** Horizontal side-to-side review slider with manual navigation triggers.
* **Contact (Get In Touch):** Direct WhatsApp messaging (`wa.me`) integration and 1-click email address copy clipboard controls.
* **Performance Enhancements:** Parallel asset loading with preconnected CDNs, non-blocking preloader, and optimized font rendering.

## 🚀 Getting Started

To run this project locally, follow these simple steps:

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/faizul-gazi/Faizul-Portfolio-.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Faizul-Portfolio-
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

The local development server will be active at `http://localhost:5173`.

### Building for Production

To compile the bundle for deployment:
```bash
npm run build
```
This compiles the production-ready code inside the `dist/` directory.

---

## 📦 Deployment on Vercel

This project is fully configured for seamless deployment on [Vercel](https://vercel.com/):

1. Link your GitHub repository to your Vercel account.
2. Choose **Vite** as the build configuration preset.
3. Vercel will automatically build the site using `npm run build` and serve the static files from the `dist` directory.
